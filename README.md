# snake-with-powerups
SNAKE. WITH. FREAKING. POWERUPS. SOOOO UNORIGINAL! Check the website to play it..

## YOUR GOAL
Eat as many apples as possible.

## Controls
Use the arrows to controll your snake
(second player: use w/a/s/d)

## Powerups
